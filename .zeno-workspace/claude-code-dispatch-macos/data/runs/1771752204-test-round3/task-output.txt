^D已更新 `greeting` 函数，添加了带默认值的 `name` 参数。

`★ Insight ─────────────────────────────────────`
- **默认参数**：`name: str = "世界"` 使用 Python 默认参数语法，调用时可省略参数
- **f-string 格式化**：`f"你好，{name}"` 是 Python 3.6+ 的字符串插值语法，比 `%` 或 `.format()` 更简洁易读
- **向后兼容**：默认参数确保现有调用 `greeting()` 仍然有效，输出 "你好，世界"
`─────────────────────────────────────────────────`

使用示例：

```python
greeting()          # 输出: 你好，世界
greeting("张三")    # 输出: 你好，张三
```
