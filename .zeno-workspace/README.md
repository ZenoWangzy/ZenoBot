## 🔍 搜索记忆系统

### QMD - 本地文档搜索引擎 ⭐ NEW

QMD 是 zenomacbot 的本地文档搜索引擎，支持：

- **关键词搜索** - 快速全文搜索
- **语义搜索** - 理解含义的搜索
- **混合搜索** - 最佳质量的搜索（BM25 + 向量 + 重排序）

### 快速搜索

```bash
# 关键词搜索
qmd search "终极目标"

# 语义搜索
qmd vsearch "关于成功的观点"

# 混合搜索（最准确）
qmd query "怎么处理记忆" -n 10 --min-score 0.3
```

## 🔄 记忆管理流程

### 新信息到来时

```
用户说了重要信息
    ↓
是否值得记录？（判断：是否长期相关）
    ↓ 是
写入 memory/2026-02-08.md（立即）
    ↓
是否足够重要？（判断：是否核心）
    ↓ 是
3-7 天后整理到 MEMORY.md
```

### 定期整理时（每 3-7 天）

```
读取最近 7 天的 memory/*.md
    ↓
筛选重要内容
    ↓
分类整理
    ↓
更新 MEMORY.md
    ↓
删除 MEMORY.md 中过期内容
    ↓
更新 QMD 索引
```
