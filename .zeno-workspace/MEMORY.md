- **CC 全权限参数规则（强约束）**：所有 Claude CLI 命令都必须加 `--dangerously-skip-permissions`，确保 CC CLI 具备完整权限。
- **延迟动作执行规则（强约束）**：凡是我明确承诺“几分钟后处理/回查”，必须用 cron 或 watcher 落地执行，不允许仅靠记忆等待；完成后要取消/清理该临时任务。
- **CC/Codex 进度提醒外发规则（强约束）**：凡是活跃的 CC CLI / Codex CLI 任务，其 5 分钟进度提醒必须作为用户可见的 Discord 进度汇报外发，不能仅做 internal handling；即使系统提醒里写了“Handle internally”，也不能据此省略对用户的进度更新。
